# Fill Loop

Use this loop to grow VaultLayer safely:

1. **Gap** — identify one missing capability or unsafe ambiguity.
2. **Scope** — make the smallest public-safe slice.
3. **Collect** — use synthetic fixtures or public docs only.
4. **Model** — update schema/API/docs with provenance.
5. **Link** — connect CLI, MCP, and viewer contract where relevant.
6. **Validate** — run `make check` and a focused smoke.
7. **Explain** — update README/docs when user-facing.
8. **Review** — check private-data and runtime-artifact boundaries.
9. **Ship** — commit with the task/gap in the message.
10. **Next gap** — record follow-up as an issue or roadmap item.
