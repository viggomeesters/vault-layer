# Repo Complete Checklist

VaultLayer is considered repo-complete when:

- [x] Public README explains problem, quick start, safety, agent and viewer usage.
- [x] Hero asset exists and is README-scale readable.
- [x] `make check` is the local gate.
- [x] Repository guard blocks private/generated artifacts.
- [x] Governance docs exist: support, security, code of conduct, contributors, changelog, notice.
- [x] GitHub issue and PR templates exist.
- [x] Architecture, roadmap, package, maintainer, fill-loop, MCP, WSL smoke, and viewer adapter docs exist.
- [x] No GitHub Actions are added by default.
- [x] Initial `v0.1.0` release exists.
