# Support

VaultLayer is an early open-source project. There is no SLA.

## Ask for help

Open a GitHub issue with:

- VaultLayer version or commit SHA.
- OS and Rust version.
- Exact command run.
- Sanitized output.
- A tiny synthetic fixture if needed.

## Do not share

Do not attach private vaults, real personal notes, generated embedding databases, credentials, tokens, Syncthing metadata, or `.obsidian` plugin state.
