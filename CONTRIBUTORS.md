# Contributors

VaultLayer is maintained by Viggo Meesters.

## Contribution model

- Public-safe contributions are welcome: code, docs, tests, adapters, and reproducible bug reports.
- Do not submit private vault content, personal notes, generated embeddings, database files, or credentials.
- Keep examples synthetic and small.

## Maintainer expectations

Maintainers should run `make check` before merging or releasing and preserve the source-of-truth rule: Markdown files stay canonical; indexes are rebuildable runtime state.
